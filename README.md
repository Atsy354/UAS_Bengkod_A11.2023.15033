# UAS Bengkel Koding Data Science

## Customer Churn Prediction

Project UAS mata kuliah Bengkel Koding Data Science mengenai prediksi customer churn menggunakan machine learning dan deployment Streamlit.

## Dataset
Sales and Marketing Dataset

## Tujuan Project
- Melakukan Exploratory Data Analysis (EDA)
- Melakukan preprocessing data
- Membangun model machine learning
- Melakukan hyperparameter tuning
- Membandingkan performa model
- Deployment model menggunakan Streamlit

## Tools & Library
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Streamlit
- Joblib
